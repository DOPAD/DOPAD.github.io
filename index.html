DOPAD : Dataset Of nanoPArticle Detection

Eric Zhonghang Qu, Andrew Matthew Jimenez, Sanat K. Kumar, and Kai Zhang

Learn more

Description

DOPAD is a dataset for nanoparticle detection in TEM images. It contains multiple TEM images in different resolutions and environments.

Overview

  • Total number of original TEM images: 72
  • Total number of images after cutting: 278,964
  • Total number of particles: 1,519,849
